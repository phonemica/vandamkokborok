# vandamkokborok
Data collected as part of a lexicostatistical analysis of Barish (Bodo-Garo) languages. 

This dataset includes four clanlects of Kokborok as spoken in Tripura, India, as well as nearly collected data on Bodo as spoken in Kokrajhar. The initial purpose of this dataset was to developed a Bayesian phylogenetic tree of the purposrted Sal language family. This study was part of the Swiss National Science Foundation _Sinergia_ project #CRSII5_183578 "Out of Asia" conducted at the Institut für Vergleichende Sprachwissenschaft, Universität Zürich.
